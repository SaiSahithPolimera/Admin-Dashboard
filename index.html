    Etch-a-Sketch

    The Etch-a-Sketch project is a web-based application that simulates the classic drawing toy. It uses Document Object Model (DOM) manipulation to dynamically create a grid of squares on a webpage. Users can "draw" on this grid by hovering their mouse over the squares, which changes their color, simulating the line drawing of the original Etch-a-Sketch. The project showcases the power of DOM manipulation in creating interactive web content.

    Threads Together App

    The "Basic Items Exchange" app is an Android application developed using Java and XML in Android Studio. It facilitates the exchange of basic items between users in close proximity. The app uses location services to connect users nearby and provides a platform for listing and requesting items. This project demonstrates the use of Android development tools and geolocation services to create a community-based sharing platform.

    Calculator

    The Basic Calculator project is a web-based application developed using HTML, CSS, and JavaScript. It provides a user-friendly interface for performing simple arithmetic operations like addition, subtraction, multiplication, and division. This project is a fun and practical way to apply and enhance web development skills, particularly in JavaScript-based DOM manipulation and event handling.
